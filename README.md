# Sorting Visualizer

A simple transformable SFML Sorting Visualizer.  
  
Every sorting algorithms are not home-made (Taken from GeeksForGeeks)  
  
Compilation under CMake 3.11
```bash
cmake . ; make ; ./sorting_visualizer
```
